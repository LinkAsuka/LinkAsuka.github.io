等高布局

在容器上显式设置了display的值为flex或inline-flex，该容器的所有子元素的高度都相等，因为容器的align-items的默认值为stretch
demo

用flex和用grid写都挺好的

  • 在实战的时候记得给内容加上高度为100%

圣杯布局

圣杯布局两边顶宽，中间自适应的三栏布局，中间栏要放在文档流前面以优先渲染

  • Flexbox布局中圣杯布局
    demo
    这波是在外层的mian加上display:flex，套个buff，然后给要排在最左边的那个标签设顺序order: -1，意思是你被强化了你先送
    然后给中间需要自适应的标签，一个flex: 1,才能在左右残血的情况下撑开全屏

那flex-direction: column;是干啥的有人知道不，是让header，main，footer竖着排序的，不然呢😏

  • Grid布局中圣杯布局

通过 grid-column 和 grid-row 来设置布局

  • 首先要给对grid的朋友科普一下
    grid-column: 1 / 2; 网格从左到右的第一个到第二个
    grid-row: 2 / 3; 网格从上倒下的第二个到第三个

正片开始
那么还是先给最外层套个buff display: grid，
设置一下上面和下面，你看这个圣杯布局| | | |中间是三个空对吧，所以有四个网格grid-column: 1 / 4给它填满😏
然后就是中间的他们都在网格的第二排到第三排让它去！！grid-row: 2 / 3
再分，先送的nav一如既往的先送grid-column: 1 / 2;然后后面的分别是2到3，3到4

  • 什么你又问grid-template是干啥的？
    grid-template: auto 1fr auto / 220px 1fr 220px;
    前面是左中右的设置，后面是上中下的设置，这可是圣杯布局中间能够自适应的关键语句

要自适应手机端的话可以一顿操作

其实还有一种方法，使用 grid-area 和 grid-template-areas属性的结合,但是我又不爱用grid，所以就不讲了反正写完了我也记不住
