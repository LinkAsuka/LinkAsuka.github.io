React 表格组件 用的最多的

在写一个后台项目的时候,用的最多的就是表格了,但是直接用 antd 给你的表格又不太好用,这个时候就要封装成你想要的样子
功能

  • 表格初始化(指进来就调一次接口)
  • 表格刷新
  • 分页 表格扩展 表格选择功能
  • 拿表格参数

组件文档

  1. 表格初始化
    request: false | 不传 | 自定义
  1. 表格刷新

refTable.current.getData()

  1. 分页 表格扩展 表格选择功能

pagination: false传了就不分页了,不传就分页
扩展

表格选择功能 rowSelection: true
获取选择参数 refTable.current.selectedRowKeys

  1. 拿表格参数
